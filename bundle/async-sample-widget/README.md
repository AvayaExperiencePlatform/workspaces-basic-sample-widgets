# Async Sample Widget
Basic LitElement web component demonstrating the Widget API calls that an Async widget might use. 
This widget can be used alongside the workspace messaging widget to see an example of messages/images/files being sent and received.

For further info, please refer to the Widget Framework Developer Documentation:
https://documentation.workspaces.avayacloud.com/widget-framework/docs/api-reference/tools-and-config/lit-element-widgets
